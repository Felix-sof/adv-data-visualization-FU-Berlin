# Advanced Data Visualization — FU Berlin 🇩🇪

Assignments and exercises from the **Advanced Data Visualization on Artificial Intelligence** course at [Freie Universität Berlin](https://www.fu-berlin.de/), taken as part of an Erasmus exchange program.

> 🎓 This is a graduate-level (PhD) Data Science course, taken as an undergraduate student during Erasmus.

## 👨‍💻 About

This repository documents my weekly progress through the course, covering modern data visualization techniques using Python and the Altair/Vega-Lite ecosystem.
